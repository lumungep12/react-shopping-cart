### REACT SHOPPING CART

### ABOUT
An online shop shopping cart made with react js

### TECHNOLOGIES USED:
    * HTML5
    * CSS3
    * Bootstrap4
    * React-bootstrap
    * Reactjs 
    * Axios
    * Redux
    * Express.js
    * mongoose db
    * react-redux, redux-thunk
    * shortid
    * react-modal, react-reveal

### FEATURES
    * Filter Items
    * Cart Logic
    * Ordering Backend Logic
